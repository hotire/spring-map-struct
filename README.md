# Map Struct

### OverView

개발할 때 흔히 POJO에서 다른 POJO로 변환하는 일이 많다. 

흔히 persistence entity 에서 VO or DTO로 변환하는 일이다. 

MapStruct은 이런 부분을 자동으로 만들어준다.

### How to use

- Mappers.getMapper(YourClass.class)

- Spring, CDI
